# 故事封面图片说明

## 📁 默认封面图片

当前系统使用以下默认封面图片（需要准备这些图片放在 public 目录下）：

- `story-cover-1.jpg` - 冒险主题封面
- `story-cover-2.jpg` - 友谊主题封面
- `story-cover-3.jpg` - 勇气主题封面
- `story-cover-4.jpg` - 探索主题封面
- `story-cover-5.jpg` - 梦想主题封面

## 📐 图片规格建议

- 宽高比：3:4（例如：600x800px）
- 格式：JPG 或 PNG
- 文件大小：建议小于 500KB
- 内容：温馨、童趣、色彩鲜艳的儿童插画风格

## 🎨 临时方案

如果暂时没有准备图片，系统会自动降级使用渐变背景色 + 图标的方式显示封面。

## 🔄 后续优化

1. 支持用户上传自定义封面
2. AI 自动生成故事封面
3. 根据故事主题智能匹配封面
